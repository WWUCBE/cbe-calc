To use this, you first need to be running a webserver in the extension directory. (That is in the CBE_Chrome_Extension directory; elsewhere won't work due to absolute paths being used for the test pages.) Easiest way is to use python's built in server:
python -m SimpleHTTPServer
Run that in a terminal, then navigate to the automated-testing.html page via the server (e.g. http://localhost:8000/testscripts/automated-testing.html).
